Jobs Types¶

class indico.types.jobs.Job(**kwargs)¶

An Asynchronous Job in the Indico Platform.

Many operations like DocumentExtraction and ModelGroupPredict are handled in the Indico Platform via Jobs. When you make a DocumentExtraction call, you are queuing a job in the Platform and a Job object is returned by client.call(..)

id¶

job id

Type

int

status¶

“SUCCESS”, “FAILURE”, “REJECTED”, “REVOKED”, “IGNORED”, “RETRY”

Type

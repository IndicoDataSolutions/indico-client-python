Model Group Types¶

class indico.types.model_group.ModelGroup(**kwargs)¶

A Model Group in the Indico Platform.

Think of Model Groups as a container for individual models that are trained each time you click the Retrain button in Review or label more data in Teach. One of these models will be the current, “selected” model. It’s usually the latest model but can also be the best performing.

id¶

The model group id

Type

int

name¶

The name of the model group

Type

str

status¶

“CREATED”, “TRAINING”, “COMPLETE” or “FAILED”

Type

str

selected model

A Model object representing the selected (active) model.

Type

Model

task_type¶

Type of machine learning task the model solves

Type

str
